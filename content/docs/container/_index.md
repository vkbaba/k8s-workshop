## ハンズオンの概要
このハンズオンでは、コンテナイメージを構築して実行する方法の基本を学びます。使用するツールはDocker です。
{{<hint info>}}
Docker 以外にも様々なコンテナランタイムがあります。

https://kubernetes.io/docs/setup/production-environment/container-runtimes/
{{</hint>}}

https://tanzu.vmware.com/developer/workshops/
